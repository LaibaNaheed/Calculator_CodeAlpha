CodeAlpha_Calculator
(Project Description)

This project is a Simple Web-Based Calculator developed using HTML, CSS, and JavaScript as part of the CodeAlpha Internship Program.

The calculator performs essential arithmetic operations and includes additional mathematical functions such as Logarithm and Square Root. It is designed with a modern lavender-themed user interface, ensuring a clean, responsive, and user-friendly experience.

This project demonstrates fundamental frontend development concepts including DOM manipulation, event handling, keyboard interaction, and dynamic UI design.

🚀 Features

➕ Addition

➖ Subtraction

✖ Multiplication

➗ Division

🧮 Square Root Function

📊 Logarithm Function

⌫ Backspace Function

🧹 Clear (Reset) Option

⌨ Keyboard Support

💜 Modern Lavender UI Design

📱 Responsive Layout

🛠️ Technologies Used:

HTML5 – Structure of the application

_RCC3 – Styling and layout design (Grid & Flexbox)

JavaScript (ES6) – Calculator logic and DOM manipulation
